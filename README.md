# xiaobot
小报童导航网站 项目地图驿站 小报童精选 一个优质专栏聚合平台，为您快速找到值得付费的专业内容。一款基于AI 编程开发的小报童专栏导航，支持快速构建自己的小报童导航网站，获取用户购买小报童专栏后的返利。 本项目默认文件已带了数据库和专栏数据（含200+小报童专栏），可以直接快速部署 在线预览地址：https://xmdtyz.com 本项目支持多种部署方式，包括： 云平台部署：Vercel、Netlify、Railway、Render 静态部署：EdgeOne Pages、GitHub Pages
